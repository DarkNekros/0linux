Aide rapide concernant votre système
____________________________________

Tout d'abord, un peu d'aide concernant l'aide elle-même. Lorsque vous 
rencontrez une visionneuse de texte telle que celle que vous êtes en train 
de lire pendant l'installation, sachez que vous pouvez vous déplacer dans 
le document comme suit :

Pg. SUIV. ou ESPACE - Descendre d'une page
PG. PRÉC. OU B      - Remonter d'une page
ENTRÉE ou BAS ou J  - Descendre d'une ligne
HAUT ou K           - Remonter d'un ligne
GAUCHE ou H         - Se déplacer vers la gauche
DROITE ou L         - Se déplacer vers la droite
0                   - Aller en debut de ligne
DÉBUT/HOME          - Aller au debut du fichier
FIN/END             - Aller a la fin du fichier
/                   - Rechercher vers le bas
?                   - Rechercher vers le haut
N                   - Répéter la dernière recherche (vers le bas)

Les consoles virtuelles
-----------------------

Sachez que vous utilisez maintenant un système multi-tâches.
Vous n'êtes donc pas confiné au programme d'installation. Vous pouvez 
vous connecter sur d'autres consoles (ou « terminaux ») et voir ce qui 
se passe à n'importe quel moment sans bloquer le processus d'installation. 
Pour cela, il vous faudra apprendre les commandes de la console sous Linux. 
Vous utiliserez souvent ces commandes en utilisant Linux.

Les consoles virtuelles et l'historique : l'écran que vous regardez en ce 
moment même est probablement la console virtuelle numéro 1 (« /dev/tty1 » 
en jargon Linux). Il y a habituellement plusieurs consoles virtuelles 
disponibles. 

Lorsque vous vous connectez via le  média d'installation, il y a plusieurs 
consoles activées par défaut. Pour basculer de l'une à l'autre, utilisez 
les touches Alt+F1, Alt+F2, Alt+F3 et ainsi de suite pour choisir quelle 
console utiliser. Pendant l'utilisation d'une console, une mémoire pour 
votre historique vous est allouée. Pour naviguer en arrière dans cet 
historique, maintenez enfoncée la touche MAJ. DROITE et appuyez sur 
PG. PRÉC., vous remonterez dans les écrans précédents. Pour naviguer 
plus avant, appuyez sur PG. SUIV. tout en maintenant la touche 
MAJ. DROITE enfoncée. Cela peut s'avérer fort utile pour lire les 
messages lors du demarrage, lesquels défilent souvent trop rapidement 
à l'écran. 

Sur le média d'installation, les trois premieres consoles sont des 
consoles de connexion (« login »). La quatrième console est utilisée pour 
l'affichage des messages du système durant l'installation, tels que 
l'état d'avancement d'un formatage, les messages du noyau, etc.


Déroulement de l'installation
-----------------------------

1- Partitionnement du disque dur 

Vous devez vous assurer que votre disque dur a été préparé pour accueillir 
un système Linux. L'installateur ne le fera pas pour vous.

Il vous faut au minimum une partition de type Linux et optionnellement une 
partition d'échange (ou partition « swap »). 

Si vous disposez de suffisamment d'espace disque, il est plus 
aisé de gérer un système partitionné de la facon suivante  :

	+ 5 giga-octets minimum pour la racine du système (« / ») 
	+ l'espace dédié aux utilisateurs sur '/home' 
	+ 1 partition « swap » (un peu plus que votre quantité de memoire RAM)

Faire une partition séparée pour /home est optionnel mais recommandé, celle-ci
contenant entre autres les paramètres et préférences des utilisateurs.

Le partitionnement s'effectue avec un outil de partitionnement tel que 
le programme 'cfdisk', ou bien avec sa version plus « vieille école », 
'fdisk'. Les deux programmes font le même travail.

Vous allez avoir besoin d'une partition de type 83 (type Linux par défaut) 
et optionnnellement une de type 82 (la partition d'echange ou « swap ») afin 
que l'installateur reconnaisse vos partitions comme exploitables.

	a) Si vous choisissez 'cfdisk', les menus vous guideront.

	b) Si vous choisissez 'fdisk', utilisez la commande 't' pour choisir le 
	   type des partitions puis finissez par la commande 'w' pour écrire la 
	   nouvelle table de partitions avant de quitter.

Par defaut, 'fdisk' et 'cfdisk' sélectionneront le premier lecteur de
disque dur de votre machine. Sur les machines équipées en IDE, le disque
sera /dev/hda. Pour celles équipées en SATA ou SCSI, ce sera /dev/sda. 
Pour partitionner des disque durs supplémentaires, vous devrez les indiquer. 

Exemples :

Pour partitionner le second disque dur IDE avec 'cfdisk', tapez :

	# cfdisk /dev/hdb

Pour partitionner le second disque dur SATA ou SCSI avec 'cfdisk', tapez :

	# cfdisk /dev/sdb

2- Lancement de l'installateur

Après avoir tapé 'installateur', suivez simplement les rubriques du haut 
vers le bas. Si vous faites les choses dans l'ordre, l'installateur vous 
mènera aux rubriques suivantes automatiquement. 

J'espère que les rubriques parlent d'elles-mêmes. Lisez attentivement ce
qui est inscrit a l'écran et tout devrait bien se passer.

Quand l'installation et la configuration sont terminées, choisissez 'Quitter' 
dans le menu principal pour redémarrer sous votre nouveau système Linux.
